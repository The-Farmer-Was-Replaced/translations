@button_tooltip_load = Lade oder erstelle einen anderen Spielstand.
@button_tooltip_save = Speichert das Spiel.
@button_tooltip_options = Öffnet das Optionsmenü.
@button_tooltip_discord = Trete dem Discord bei, um mit anderen Spielern und dem Entwickler zu chatten, Fehler zu melden oder Hilfe zu bekommen.
@button_tooltip_quit = Schließt das Spiel ohne zu speichern.
@button_tooltip_execute = Startet oder stoppt die Ausführung des Codes in diesem Fenster.
@button_tooltip_step_by_step = Führt einen Schritt des Codes aus.
@button_tooltip_new_function = Erstellt ein neues Fenster.
@button_tooltip_minimize = Minimiert dieses Fenster.
@button_tooltip_delete = Löscht die Datei in diesem Fenster dauerhaft.
@button_tooltip_table_of_contents = Inhaltsverzeichnis
@option_tooltip_graphics = Beeinflusst verschiedene Renderpipeline-Einstellungen wie Schatten, Umgebungsverdeckung und Bloom.
@option_tooltip_vfx_limit = Begrenzt die Anzahl der visuellen Effekte wie Partikeleffekte und den Gegenstandsaufnahmeeffekt, die pro Sekunde abgespielt werden können.
@option_tooltip_volume = Passt die Lautstärke aller Soundeffekte an.
@option_tooltip_volume_damping = Die Lautstärke wird für jeden gleichzeitig abgespielten Soundeffekt mit diesem Faktor multipliziert, damit deine Ohren bei hohen Drohnengeschwindigkeiten nicht explodieren.
@option_tooltip_sfx_limit = Begrenzt die Anzahl der Sounds, die pro Sekunde abgespielt werden können.
@option_tooltip_autosave = Speichert das Spiel automatisch alle 30 Sekunden, wenn aktiviert. Kann nicht aktiviert werden, während der Dateiwächter aktiviert ist.
@option_tooltip_file_watcher = Wendet automatisch Änderungen an den Code-Dateien im Speicherordner an, sodass ein externer Code-Editor verwendet werden kann. Kann nicht aktiviert werden, während Autosaves aktiviert sind.
@option_tooltip_frames = Begrenzt die FPS des Spiels. Vsync synchronisiert die FPS mit der Bildwiederholrate deines Monitors.
@option_tooltip_viewport_sliding = Deaktiviert das Gleiten des Viewports, wenn du ziehst und loslässt.
@option_tooltip_UI_size = Ändert die Größe der UI-Elemente.
@option_tooltip_tabs_to_spaces = Wandelt alle Tabs in 4 Leerzeichen um.
@option_tooltip_error_forgot_call = Wenn aktiviert, wirft das Spiel einen Schutzfehler, wenn du versuchst, die Wahrhaftigkeit eines Funktionsobjekts zu evaluieren, da dies normalerweise bedeutet, dass du die `()` vergessen hast.
@option_tooltip_error_shadow = Wenn aktiviert, wirft das Spiel einen Schutzfehler, wenn es eine lokale Variable gibt, die denselben Namen wie eine globale Variable hat.
@option_tooltip_print_warnings = Gibt Warnungen über verschiedene Dinge an die Ausgabe aus, wie z.B. nicht genügend Gegenstände für einen Handel zu haben.
@keybind_tooltip_indent_selection = Fügt allen ausgewählten Zeilen eine Einrückung hinzu. Dies muss eine Taste sein, die ein Zeichen einfügt. Kombinationen mit Strg oder Alt funktionieren daher nicht.
@keybind_tooltip_unindent_selection = Entfernt die Einrückung von allen ausgewählten Zeilen. Dies muss eine Taste sein, die ein Zeichen einfügt. Kombinationen mit Strg oder Alt funktionieren daher nicht.
@keybind_tooltip_menu = Öffnet oder schließt das Hauptmenü.
@keybind_tooltip_move_down = Bewegt die Kamera nach unten.
@keybind_tooltip_move_left = Bewegt die Kamera nach links.
@keybind_tooltip_move_right = Bewegt die Kamera nach rechts.
@keybind_tooltip_move_up = Bewegt die Kamera nach oben.
@keybind_tooltip_redo = Wiederholt das letzte Rückgängigmachen.
@keybind_tooltip_save = Speichert das Spiel.
@keybind_tooltip_undo = Macht die letzte Änderung rückgängig.
@keybind_tooltip_start_execution = Startet die Ausführung des zuletzt aktiven Fensters oder setzt sie fort, wenn sie pausiert ist.
@keybind_tooltip_stop_execution = Stoppt die Ausführung.
@keybind_tooltip_next_step = Führt einen Schritt aus, wenn die Ausführung pausiert ist.
@keybind_tooltip_pause_execution = Pausiert die Ausführung.
